Privacy Policy

For Rainbow Tools - Image & File Toolkit

Last Updated: December 1, 2023

This privacy policy describes how Media Ghor ("we", "our", or "us") collects, uses, and shares information about you when you use our Android application "Rainbow Tools - Image & File Toolkit" (the "App").

Introduction

At Media Ghor, we are committed to protecting your privacy and being transparent about our data practices. This Privacy Policy explains how Rainbow Tools handles your information when you use our application for image processing and file management.

Rainbow Tools is designed to process your images and files while respecting your privacy. We implement measures to ensure that your data is handled securely and in accordance with this policy.

Our Privacy Commitment for Rainbow Tools

No Personal Data Collection
Files Deleted After Processing
Encrypted Data Transmission
No Unnecessary Data Access

Android Permissions Explained

Rainbow Tools requires certain permissions to function properly. Here's a detailed explanation of why each permission is needed and how it's used:

INTERNET android.permission.INTERNET

Purpose: This permission allows Rainbow Tools to connect to the internet to provide advanced features.

  • AI-powered image processing (background removal, enhancement)
  • Text extraction using OCR technology
  • File format conversion services
  • Downloading necessary updates and resources

Data Handling: All internet communications are encrypted using SSL/TLS protocols.

ACCESS_WIFI_STATE android.permission.ACCESS_WIFI_STATE

Purpose: This permission helps optimize the app's performance based on network conditions.

  • Detect when you're on Wi-Fi to enable larger file processing
  • Provide appropriate feedback about network availability
  • Optimize data usage when on mobile networks

Data Handling: We do not collect information about specific networks you connect to.

READ_EXTERNAL_STORAGE android.permission.READ_EXTERNAL_STORAGE

Purpose: This permission allows Rainbow Tools to access images and files you select for processing.

  • Access images you choose to enhance or modify
  • Read documents you select for text extraction or conversion
  • Browse your gallery to select files for processing

Data Handling: We only access files you explicitly choose. We do not scan or access other files on your device.

WRITE_EXTERNAL_STORAGE android.permission.WRITE_EXTERNAL_STORAGE

Purpose: This permission allows Rainbow Tools to save processed files to your device.

  • Save enhanced images to your gallery
  • Store converted files in your preferred location
  • Create output folders for organized file management

Data Handling: We only write files to locations you approve and never modify existing files without your consent.

READ_MEDIA_IMAGES android.permission.READ_MEDIA_IMAGES

Purpose: This is the updated permission for Android 13+ that replaces READ_EXTERNAL_STORAGE for media access.

  • Access photos and images for processing
  • Select images from your gallery for enhancement
  • Choose images for background removal or text extraction

Data Handling: This permission provides more granular control and only grants access to images, not other file types.

File Processing Details

How We Handle Your Files

When you process images or files through Rainbow Tools:

  • Local Processing: Simple operations are processed directly on your device
  • Cloud Processing: Advanced AI features (background removal, OCR) may process files on our secure servers
  • Temporary Storage: Files sent to our servers are automatically deleted after processing (typically within 1 hour)
  • No Permanent Storage: We do not permanently store your processed files on our servers
  • Encryption: All file transfers are encrypted using industry-standard protocols

How We Use Information

Any information we collect is used solely for the purpose of providing and improving the Rainbow Tools application:

  • To provide image processing services (enhancement, background removal, etc.)
  • To enable file conversion between different formats
  • To extract text from images using OCR technology
  • To optimize app performance and user experience
  • To fix bugs and improve functionality through anonymous crash reports
  • To respond to user support requests when voluntarily contacted

AI Processing Note

For AI-powered features like background removal and image enhancement, your images are processed using machine learning algorithms. These algorithms analyze the visual content of your images but do not store or remember specific images after processing is complete.

Data Sharing and Disclosure

We do not sell, trade, or otherwise transfer your personal information or file contents to third parties. The limited circumstances where information may be shared include:

  • Service Providers: Trusted third parties who assist us in operating our app (e.g., cloud hosting, AI processing services), subject to strict data protection obligations
  • Legal Requirements: If required by law or in response to valid legal process
  • Business Transfers: In connection with a merger, acquisition, or sale of assets

No Advertising Networks

Rainbow Tools does not integrate with advertising networks and does not use your data for targeted advertising.

Third-Party Services

We may use third-party services for specific functionalities (e.g., OCR processing, cloud infrastructure). These services are bound by contractual obligations to handle your data securely and only for the purposes we specify.

Data Security

We implement appropriate technical and organizational measures to protect any information we process:

Our Security Measures

  • Encryption: All data transmissions are encrypted using SSL/TLS protocols
  • Secure Servers: Our processing servers are secured with industry-standard protections
  • Access Controls: Strict access controls limit who can access processing systems
  • Regular Audits: We regularly review and update our security practices

Security Limitations

While we implement robust security measures, no method of transmission over the Internet or electronic storage is 100% secure. We cannot guarantee absolute security but we continuously work to maintain and improve our security protocols.
